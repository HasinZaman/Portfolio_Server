ResponseStatusCode is a enum of all HTTP response status codes

Variants between 100 - 199 relate to Information codes Variants between 200 - 299 relate to Successful codes Variants between 300 - 399 relate to Redirection codes Variants between 400 - 499 relate to Client Error codes Variants between 500 - 599 relate to Server Error codes
